# Monthly Budget Cycle Checklist

Aligned with the Holdings financial reporting structure. CFO runs this on the first business day of each month.

## Pre-Review (Day 1-2)

- [ ] Pull cost summary for completed month
- [ ] Pull per-agent costs for completed month
- [ ] Pull per-project costs for completed month
- [ ] Compare against budget allocations from governance context
- [ ] Calculate utilization percentages per agent and project
- [ ] Identify any agents that hit budget limits during the month
- [ ] Identify any agents consistently under-utilizing budget

## Analysis (Day 2-3)

- [ ] Run trend analysis (month-over-month comparison)
- [ ] Calculate projected next-month spend based on trends
- [ ] Identify rebalancing opportunities
- [ ] Score each rebalancing recommendation with VFM (from decision-protocol)
- [ ] Draft budget section for board report

## Reporting (Day 3-5)

- [ ] Write monthly financial report as issue document
- [ ] Include: executive summary, per-agent breakdown, per-project breakdown, recommendations
- [ ] Submit to Chief of Staff for board report assembly
- [ ] If rebalancing recommended, submit as separate decision with escalation path

## Board Package Financial Section

The CFO produces these items for the monthly board package:

1. **Executive dashboard:** Total spend, budget remaining, top 3 variances
2. **Entity P&L equivalent:** Per-project spend breakdown with budget comparison
3. **Forecast:** Projected spend for next month based on current trends
4. **Recommendations:** Any budget actions needed with VFM scores

# Governance Setup

This project contains the onboarding tasks that must run when the governance package is first imported. Complete these tasks in order.

## Tasks

1. **Setup Governance Context** — Chief of Staff fills out the company governance context document
2. **Initial Financial Review** — CFO runs first financial analysis and budget review
3. **Initial Health Check** — COO runs first operational health scan and compliance review

# Setup Governance Context

Use the `company-governance-context` skill to create the governance context for this company.

## What You Need

1. Read the company description and goals from the Paperclip API
2. Ask the human board owner to confirm or provide:
   - Company mission
   - Strategic goals
   - Risk appetite
   - Compliance requirements
   - Decision authority thresholds
   - Escalation preferences
   - Reporting cadence
3. Write the governance context document using the template from the skill's references

## Done When

- Governance context issue document exists with all sections filled in
- Human board owner has confirmed the context is accurate
- CFO and COO can read the context and begin their reviews
